# PhishAware - Tools & Technologies Used

## Backend
- **Framework**: Flask 3.0.0
  - Python web framework for building the application server
  - Handles routing, templating, and request/response management
  
- **Server**: Gunicorn 21.2.0
  - WSGI HTTP Server for production deployment
  - Supports multi-worker processes for scalability

- **Templating**: Jinja2 3.1.2
  - Template engine for rendering dynamic HTML pages
  - Used for server-side rendering of quiz data and page content

- **Environment**: Python 3.x
  - Programming language for backend logic
  - Supports JSON parsing and file operations

## Frontend
- **HTML5**: Semantic markup for page structure
  - Responsive viewport configuration
  - Accessibility features

- **CSS3**: Advanced styling and animations
  - CSS Grid for responsive layouts
  - CSS Flexbox for component alignment
  - CSS Variables (Custom Properties) for theming
  - Media queries for mobile responsiveness
  - Animations and transitions for visual effects

- **JavaScript**: Client-side interactivity
  - Dynamic form handling
  - Quiz validation and scoring
  - Interactive simulation feedback

## Utilities & Dependencies
- **Werkzeug 3.0.1**
  - WSGI utility library for Flask
  - Handles HTTP exceptions and security utilities

- **python-dotenv 1.0.0**
  - Environment variable management
  - Configuration management for sensitive data

## Data Format
- **JSON**: Data serialization format
  - Stores quiz questions and answers
  - Lightweight and easy to parse

## Development & Deployment
- **Git**: Version control system
- **Procfile**: Heroku deployment configuration
- **Requirements.txt**: Python dependency management

## Design System
- **Color Scheme**: Dark cybersecurity theme with neon green/cyan accents
  - Primary Dark: #0a0e27
  - Neon Green: #00ab11
  - Neon Cyan: #00ffff
  - Danger Red: #ff1744

- **Typography**: 
  - Font Family: Segoe UI, Courier New, Monospace
  - Sans-serif for UI, Monospace for terminal effects

- **Layout Framework**:
  - Max-width container: 1200px
  - Responsive grid system with auto-fit
  - Minimum column widths for different screen sizes

## Performance Features
- **Smooth scrolling** behavior
- **Lazy loading** optimizations
- **CSS animations** for enhanced UX
- **Responsive images** and flexible containers
- **Mobile-first** design approach
